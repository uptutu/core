# API 文档

----
## entity
1. create
```bash
post .../plugins/{plugin}/entities
```
2. delete
```bash
delete .../plugins/{plugin}/entities/{entity}
```
3. patch
```bash
patch .../plugins/{plugin}/entities/{entity}
```
4. read
```bash
get .../plugins/{plugin}/entities/{entity}
```
5. list
```bash
get .../plugins/{plugin}/entities
```
